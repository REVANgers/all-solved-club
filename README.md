# All Solved Club

코딩테스트 올솔을 목표로 하는 알고리즘 스터디입니다.

## Guideline

1. members 디렉토리 내 개인 폴더를 만들어주세요.

2. 개인 폴더의 이름을 Slack 멤버 ID와 동일하게 설정할 경우 자동 알림 설정이 됩니다. (구현 예정)
